El proyecto

Lenguaje de Restaurantes es una aplicación web para gestionar el menú de un restaurante. Permite ver, añadir, editar y eliminar platos, bebidas y postres.

Está hecho con HTML, CSS y JavaScript en el frontend, y Node.js con Express en el backend. Los datos se guardan en una base de datos MySQL.

Trabajo hecho por Ana Martos Zambrana y Aarón Martín Pozo
